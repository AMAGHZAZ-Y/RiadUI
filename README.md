## Riad UI is a mobile first minimalistic UI for the Svelte framework.

### Features: 
- Written in TypeScript
- Accessible with WAI-ARIA standards
- Dark Theme API
- Highly customizable
- Easy configuration
- SvelteKit and SSR Compatible
